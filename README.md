LiveMood is a real-time facial emotion detection mini project that uses your webcam to detect a face and display the dominant emotion live.

## Features
- Real-time webcam feed
- Face detection with rectangle
- Emotion detection (happy, sad, angry, neutral, etc.)
- Color-coded emotion labels

## Tech Stack
- Python 3.10
- OpenCV
- DeepFace
- TensorFlow / tf-keras

## How to Run
pip install opencv-python numpy deepface tf-keras  
python live_mood.py
